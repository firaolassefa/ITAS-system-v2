## Question Format (Copy This!)

```
Question 1: [Your question text]
Type: Practice
A) [First option]
B) [Second option]
C) [Third option] (optional)
D) [Fourth option] (optional)
Correct Answer: A
Explanation: [Why this is correct]
Points: 5

Question 2: [Your question text]
Type: Exam
A) [First option]
B) [Second option]
Correct Answer: B
Explanation: [Why this is correct]
Points: 20
```

**Rules:**
- `Type: Practice` = W3Schools style (unlimited tries)
- `Type: Exam` = For certificate (limited tries)
- `A)`, `B)`, `C)`, `D)` = Answer options
- `Correct Answer: A` = Which one is correct
- `Explanation:` = Why it's correct (optional but recommended)
- `Points:` = How many points (optional, default 1)

### Module Quiz

- **Must pass** to unlock next module
- **See results** after submit
- **Can retry** if fail
- **Passing score**: 70%

### Final Exam
- **Timed** (60 minutes)
- **3 attempts** maximum
- **Must pass** to get certificate
- **No answer preview** during exam
- **Passing score**: 80%
- **Certificate** auto-generated if pass

## Troubleshooting

### "401 Unauthorized" when uploading
**Fix:** Logout and login again as `contentadmin` / `Content@123`

### Questions uploaded but no answers
**Fix:** Check format:
- Use `A)` not `1)`
- Add `Correct Answer: A` line
- Have at least 2 options

### Practice questions not showing instant feedback
**Fix:** Make sure you added `Type: Practice` in the file
