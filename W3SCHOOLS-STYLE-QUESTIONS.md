## Format Rules

### REQUIRED Fields:

1. **Question Number and Text**
   ```
   Question 1: What is VAT?
   ```

2. **Type** (NEW! Like W3Schools)
   ```
   Type: Practice
   ```
   OR
   ```
   Type: Exam
   ```
   - **Practice** = Unlimited tries, see answers (W3Schools style)
   - **Exam** = For certificate, limited tries
   - If not specified, defaults to **Exam**

3. **Answer Options** (2-4 options)
   ```
   A) Value Added Tax
   B) Variable Tax
   ```

4. **Correct Answer**
   ```
   Correct Answer: A
   ```

### OPTIONAL Fields:

5. **Explanation** (Recommended)
   ```
   Explanation: VAT stands for Value Added Tax
   ```

6. **Points**
   ```
   Points: 10
   ```

## Troubleshooting

### Problem: 401 Unauthorized

**Solutions:**
1. **Logout and login again**
   - Session may have expired
   - Token is valid for 8 hours

2. **Check your role**
   - Login as: `contentadmin` / `Content@123`
   - Or: `trainingadmin` / `Training@123`
   - Or: `systemadmin` / `Admin@123`

3. **Clear browser cache**
   - Press Ctrl+Shift+Delete
   - Clear cookies and cache
   - Login again

### Problem: Questions uploaded but no answers

**Check your format:**
```
✅ CORRECT:
A) Value Added Tax
B) Variable Tax
Correct Answer: A

❌ WRONG:
1) Value Added Tax  (should be A))
2) Variable Tax     (should be B))
Answer: A           (should be "Correct Answer: A")
```

### Problem: All questions are Exam type

**Add Type field:**
```
Question 1: What is VAT?
Type: Practice  ← ADD THIS
A) Value Added Tax
```
