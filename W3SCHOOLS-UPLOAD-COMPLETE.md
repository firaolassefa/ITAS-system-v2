## How to Use

### Step 1: Create Your File

Copy `SAMPLE-VAT-QUESTIONS.txt` to Word and save as `.docx`

OR create your own with this format:

```
Question 1: What is VAT?
Type: Practice
A) Value Added Tax
B) Variable Tax
Correct Answer: A
Explanation: VAT is Value Added Tax
Points: 5

Question 2: What is the VAT rate?
Type: Exam
A) 10%
B) 15%
Correct Answer: B
Points: 20
```

### Step 2: Upload

1. **Login**: `contentadmin` / `Content@123`
2. **Go to**: Question Management
3. **Select**: Course and Module
4. **Click**: "Upload File"
5. **Choose**: Your .docx or .pdf file
6. **Click**: "Upload & Import"

### Step 3: Done!

```
✅ Successfully imported 10 questions!
- 5 Practice questions (W3Schools style)
- 5 Exam questions (for certificate)
```
